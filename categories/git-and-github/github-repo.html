How to add your local project to GitHub

A developers guide to get started.

  1. Start a new repository on GitHub:

    You need to log in to or create a user on github.com. Then under 'Dashboard' press the button "New". Choose a name for your new repository (I like to name the project the same as my local project). Select if you want your project to be Public or Private. You can add a description if you want. Then press 'Create a new repository'

  2. Add a git repository to your local project using HTTPS:

    Open terminal and navigate to your project folder where the pom.xml file is located. Create a README.md file (if you don't allready have one). 

    $ echo "# myprojectname" >> README.md

    Then initialize a git project

    $ git init

    Add (stage) all files

    $ git add .

    (You can also just add the README.md file)

    $ git add README.md

    Commit file(s) and write a commit message

    $ git commit -m "first commit"

    Create a main branch

    $ git branch -M main

    Add your local files to your remote repository on GitHub

    $ git remote add origin https://github.com/username/myprojectname.git

    And lastly push your changes

    $ git push -u origin main

    If you refresh GitHub.com you can now se your project.
