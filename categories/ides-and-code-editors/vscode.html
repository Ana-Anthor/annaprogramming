Visual Studio Code (VSCode)

Last updated: 15 April 2026

About

VSCode is my absolute favourite Code Editor because it is easy to use and has a lot of customisation options and plugins. You can really make this editor feel like your own. The code editor is opensource and is created by Microsoft. It is popular especially among web developers.

Useful shortcuts

  • ! + Tab

    After creating a html file you can type "! + Tab" in the file and it automatically writes the html template:

    
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        
    </body>
</html>
